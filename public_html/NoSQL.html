NoSQL

  • NoSQL stands for Not Only. This terminology can be used to refer to a variety of different database technologies
    • Key-value pairs
    • Graph-based databases, which specialise in representing and handling problems from graph theory
    • Document databases which are used for analysing documents and other texts
    • Geographic Information Systems (GIS), which are used for geographic analysis
  • These types of databases are often specialised for particular functions
  • These technologies are more complementary technologies to relational databases rather than replacement technologies
  • SQL and the underlying relational database technology increasingly support functionality similar to NoSQL Databases

Key-value pairs

  • Key-value pairs are where the columns of data can vary between rows and the columns can contain lists of things
  • Key-value pair databases provide excellent performance in a web environment for managing data about online sessions
  • Key Value databases are often used on a website in conjuction with data warehouses that support more structured information
