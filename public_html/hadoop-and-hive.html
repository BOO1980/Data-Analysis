Hadoop

  • Hadoop is one of the technologies highly associated with big data
  • Hadoop is conjuction with MapReduce
  • Hadoop is an open-source project and the code is available for free online
  • The goal of Hadoop is to have a framework for reliable, scalable, distributed computing
  • Hadoop is a platform for processing humongous amounts of data
  • Hadoop particularly processes data from sources such as web logs, high energy physics, high volumes of streaming images and other voluminous data sources
  • Google developed a parallel framework around MapReduce, and now its a framework for programming data intense tasks on large grid computers
  • Hadoop actually has a family of technologies and MapReduce is only one application
  • Hadoop have other tools with colourful names such as
    • Hive
    • Mahout
    • Cassandra
    • Pig
  • Although the underlying technology is different from relational databases there are similarities in the problems these technologies are trying to solve
  • Hadoop languages are based on SQL syntax. Such as CQL
  • Hive in particular is being developed into fully functional SQL engine and can run many queries
